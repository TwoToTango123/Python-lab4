## Задание - M3

# Что умеет этот калькулятор?

## Поддерживаемые операции:
- Базовые: + (сложение), - (вычитание), * (умножение), / (деление)
- Степень: ** (возведение в степень)
- Целочисленные операции: // (целочисленное деление), % (остаток от деления)
- Унарные операторы: + и - перед числами (например: -5, +10)

## Основные файлы и функции:

main.py - главный файл
calc.py - файл с основными исполнительными функциями:

1) tokenize(text) - разбивает выражение на части (токены)

- Проверяет правильность выражения
- Находит ошибки ввода
- Разделяет числа, операторы и скобки

2) calculations(tokens) - главная функция вычисления

- Принимает строку с выражением
- Возвращает результат

## Проверки ошибок:

#### Калькулятор проверяет большинство возможных ошибок:

- Деление на ноль
- Пропущенные операторы
- Недопустимые символы
- Числа с ведущими нулями
- Неправильные последовательности операторов

## Особенности

1) Правильно обрабатывает приоритет операций
2) Поддерживает вложенные скобки
3) Показывает понятные сообщения об ошибках
4) Работает с дробными и целыми числами
5) Проверяет целочисленные операции (// и % только для целых чисел)

## Допущения:

- При целочисленном делении или остатке от деления дробного числа на единицу ошибка (Операции // и % допустимы только для целых чисел)
- Проблема точности чисел с плавающей точкой (Например: если ввести 0.11-0.09 ответ получится 0.0200000000004)
